## Lessons Learned

- **JSX parser gotcha**: `map((x) => ({...}))` — `({` = object literal, ไม่ใช่ JSX expression block. ใช้ `map((x) => ...)` หรือ `map((x) => { return <A/>; })` แทน
- **React component must tolerate API shape changes** — defense in depth: `Array.isArray()` check + `?? []` fallback
- **maw config fields มีผลต่อ UI** — `node` field ถูกใช้โดย `useFederationList` hook เพื่อ gate availability
- **maw serve reload**: การแก้ `~/.config/maw/maw.config.json` ต้อง kill + restart maw server (ไม่มี hot reload)
