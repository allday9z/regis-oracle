## Lessons Learned

- maw canonical oracle name = `<name>-oracle` (ไม่ใช่ bare name)
- qmd ต้องการ bun.lock ในpackage dir เพื่อใช้ bun runtime (ไม่ใช่ node)
- ghq root ควร set เป็น `/home/<user>` แล้ว repos อยู่ที่ `<root>/github.com/<org>/<repo>`
- tmux custom socket (`-L <name>`) มีเสถียรภาพกว่าใน Claude Code sandbox
- fleet config ต้องการ `project_repos: ["<org>/<repo>"]` และ windows section
