# Steps 9-10 — 2026-04-17

## Fleet ที่ตั้งค่าแล้ว

- **regis** — Oracle หลัก, fleet orchestrator (allday9z/regis-oracle)
- **herald** — Oracle ที่ 2, budded from regis (allday9z/herald-oracle)

## Commands ที่ใช้ได้

```bash
maw oracle ls          # ดู fleet
maw oracle scan        # scan หา oracle ใหม่
maw fleet health       # health check ทั้ง fleet
maw capture regis      # ดู regis กำลังทำอะไร
maw capture herald     # ดู herald กำลังทำอะไร
maw soul-sync regis    # sync memory จาก regis
maw transport          # status: tmux + nanoclaw connected
maw wake <name>        # เปิด oracle
maw sleep <name>       # ปิด oracle
```

## Transport Layers (Step 10)

- tmux: connected (local session)
- nanoclaw: connected (in-process)
- lora: disconnected (no hardware)

## สิ่งที่เรียนรู้

- Oracle ต้องชื่อ `<name>-oracle` (canonical naming)
- ghq root = /home/m2 → repos อยู่ที่ `/home/m2/<org>/<repo>`
- fleet config อยู่ที่ ~/.config/maw/fleet/
- maw serve ต้องรันก่อนใช้ API features
